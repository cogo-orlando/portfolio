Statut Terminé

NetflixLight est une plateforme de streaming moderne et immersive conçue pour offrir une expérience utilisateur fluide et dynamique. Ce projet est né de la volonté de recréer une interface complexe de type "Single Page Application" (SPA) en s'appuyant sur des données réelles et massives provenant de l'industrie cinématographique. L'objectif principal était de relever le défi technique de l'intégration d'API tierces tout en garantissant une performance optimale et une interface visuellement percutante, fidèle aux standards du web actuel. Le projet est actuellement entièrement fonctionnel et déployé sur Render.

// Apprentissages

Ce que j'ai appris

01

Intégration d'API REST

Consommer une API tierce avec fetch, gérer les réponses JSON, les erreurs et les cas limites. Comprendre la pagination et les requêtes asynchrones en JavaScript.

02

Architecture SPA

Construire une Single Page Application sans framework — routing côté client, gestion de l'état, mise à jour dynamique du DOM sans rechargement de page.

03

Serveur Node.js

Mettre en place un serveur JavaScript pour servir l'application et gérer les routes. Déploiement sur Render et gestion des variables d'environnement.

04

UI/UX avancé

Recréer une interface complexe type Netflix — recherche fluide, catégories, favoris, système de compte. Attention portée à l'expérience utilisateur et aux transitions.
