Power4 est une application web interactive reproduisant le célèbre jeu du Puissance 4, entièrement développée en HTML et CSS sans utilisation de JavaScript. L'objectif principal était de repousser les limites du CSS en simulant une logique applicative complexe habituellement gérée par des langages dynamiques. Le joueur peut interagir avec la grille, déposer ses pions et suivre le déroulement de la partie sans rechargement de page ni script externe. L'ensemble repose sur une architecture purement déclarative, mettant en avant une maîtrise approfondie du CSS et de ses possibilités cachées.

// Apprentissages

Ce que j'ai appris

01

CSS sans JavaScript

Utiliser les pseudo-classes :checked, :hover et les sélecteurs CSS pour simuler de la logique applicative — états, transitions, interactions — sans une seule ligne de JS.

02

Architecture CSS avancée

Organiser un CSS complexe pour un jeu interactif — variables, calc(), grid, flexbox. Comprendre les limites et possibilités cachées du langage.

03

Serveur Go minimal

Mettre en place un serveur HTTP Go pour servir une application statique — routing simple, gestion des fichiers statiques, déploiement sur Render.

04

Penser différemment

Résoudre un problème logique sans outil habituel — contrainte créative qui force à explorer des solutions non conventionnelles et à comprendre profondément le fonctionnement du navigateur.
