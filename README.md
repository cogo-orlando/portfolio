🛡️ Orlando.cogo | Portfolio
Bienvenue dans le dépôt de mon portfolio personnel. Conçu avec une esthétique terminal/CLI, ce site présente mon parcours, mes compétences techniques et mes projets en informatique et cybersécurité.

🚀 Aperçu
Le site est une application web moderne reflétant ma passion pour l'administration système et la sécurité offensive/défensive.

🌐 Live Demo : orlando.cogo (ou ton lien réel)

🎯 Statut : Étudiant en Bachelor Informatique (Spécialisation Cybersécurité)

📍 Focus : Sécurité des systèmes et des réseaux, exploration de vulnérabilités.

🛠️ Stack Technique
Frontend & Design
Framework : React.js / Next.js (ou préciser si c'est du HTML/CSS pur)

Styling : Tailwind CSS (pour le look sombre/néon)

Animations : Framer Motion / GSAP

Compétences mises en avant
Langages : Go, JavaScript, SQL

Systèmes : Linux (Hardening, Scripting Bash)

Sécurité : Analyse de vulnérabilités, Pentesting, CTF

📂 Structure du Projet
Le site est découpé en plusieurs sections clés, accessibles via le menu de navigation :

00 ACCUEIL : Page principale avec l'identité visuelle.

01 À PROPOS : Détails sur ma reconversion et mon parcours.

02 COMPÉTENCES : Ma stack technologique détaillée.

03 PROJETS : Galerie de mes réalisations techniques.

04 CONTACT : Liens vers GitHub, LinkedIn et Email.

05 CV : Accès direct à mon curriculum vitae.

06 STATUS : État actuel de mes recherches (Open to work).

⚙️ Installation (Local)
Si vous souhaitez explorer le code ou faire tourner le projet localement :

Cloner le dépôt

Bash
git clone https://github.com/ton-username/portfolio-orlando.git
Installer les dépendances

Bash
npm install
Lancer le serveur de développement

Bash
npm run dev
