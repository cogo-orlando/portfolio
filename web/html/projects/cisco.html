03 //

Architecture technique

Routage statique

Routes statiques configurées sur chaque routeur de site vers Router0 (hub central). Interconnexion WAN via sous-réseaux 10.0.0.x dédiés.

VLANs & Segmentation

10 VLANs répartis sur les 4 sites (VLAN10 à VLAN100). Routage inter-VLAN assuré par les switchs L3 (3650-24PS).

DHCP sur Switch L3

Service DHCP configuré directement sur les switchs Layer 3. Chaque VLAN dispose de son propre pool DHCP.

DNS & Serveur Web

Serveur DNS (192.168.100.10) et serveur Web (192.168.100.20) hébergés sur le site de Stockholm. Accessibles depuis tous les sites.

Redondance WiFi

Chaque point d'accès WiFi est doublé par un AP redondant. En cas de défaillance, la connectivité est maintenue via l'AP de secours.

Interconnexion WAN

Architecture hub-and-spoke avec Router0 comme nœud central. 4 liaisons WAN dédiées entre chaque site et le routeur central.
