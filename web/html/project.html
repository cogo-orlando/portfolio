02 Go · API · HTML · CSS

Groupie Tracker

Projet fait a deux | Intégration d'une API REST pour la récupération et l'affichage de données complexes : biographies d'artistes, calendriers de tournées et géolocalisation des salles de concert. Maîtrise des requêtes asynchrones et du traitement de flux JSON pour un affichage en temps réel.

03 Go · HTML · CSS

Power4

Projet fait seul | Conception d'un Puissance 4 utilisant uniquement les fonctionnalités natives du HTML et du CSS. L'objectif était de repousser les limites de l'intégration web en simulant une logique applicative complexe (gestion des interactions et des états) sans aucune ligne de code JS.

GitHub → Demo →
04 C#

ZOOTopie

Projet fait a deux | Développement d'un jeu de gestion au tour par tour en console. Architecture basée sur la programmation orientée objet (Héritage, Polymorphisme) pour gérer différentes espèces d'animaux, leurs besoins physiologiques et l'économie du parc. Implémentation d'une boucle de jeu logique et d'un système d'événements aléatoires.
